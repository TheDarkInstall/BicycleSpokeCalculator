# Spoke Length Calculations

- Spoke Length = SL
- Number of Spokes = NSP
- Effective Rim Diameter = ERD
- Spoke Hole Diametere = SHD
- Drive Side = DS
- Non Drive Side = NDS
- Drive Side Flange to Centre = FtCDS
- Non Drive Side Flange to Centre = FtCNDS
- Radial = R
- One Cross = 1x
- Two Cross = 2x
- Three Cross = 3x
- Four Cross = 4x
- Tension = Tens
